# **App Name**: BookBrew

## Core Features:

- Book Browsing: Allow users to browse books by genre, author, price, and ratings with advanced search and filters.
- Book Details: Provide detailed book pages with descriptions, reviews, offers, and availability.
- Personalized Recommendations: Offer book recommendations to users based on their activity and purchase history.
- Shopping Cart and Checkout: Enable users to add/remove books from their cart, proceed to checkout, and make payments via Stripe or Razorpay.
- Admin Dashboard: Provide an admin interface to manage books, view analytics, manage orders, and handle user accounts.
- Secure User Accounts: Implement JWT-based authentication and user profiles with order history and wishlist.
- Wishlist Management: Allow users to add books to a wishlist for future purchase and store wishlist data in MongoDB.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey knowledge, sophistication, and trust.
- Background color: Very light lavender (#F0F8FF), subtly desaturated from the primary, to set a professional and unobtrusive stage for content.
- Accent color: Violet (#8A2BE2), analogous to the primary, but brighter and more saturated for highlighting calls to action.
- Body font: 'PT Sans', sans-serif for body text; simple, modern and readable
- Headline font: 'Playfair', serif for headings; displays a modern but classical aesthetic.
- Use minimalist, book-themed icons with a consistent stroke weight and rounded corners for a friendly appearance.
- Implement a clean, grid-based layout with generous white space for easy content consumption.
